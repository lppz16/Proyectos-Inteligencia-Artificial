# Proyectos de Inteligencia Artificial

Este repositorio contiene los proyectos, talleres, laboratorios y actividades desarrollados durante el curso de **Inteligencia Artificial**. El objetivo es aplicar los conceptos teóricos vistos en clase mediante el desarrollo de soluciones prácticas utilizando diferentes técnicas y herramientas de IA.

## Contenido

- Talleres
- Laboratorios
- Proyectos
- Ejercicios prácticos
- Documentación y reportes

## Estructura del repositorio

```text
Proyectos-Inteligencia-Artificial/
│
├── Talleres/
├── Quices/
├── Proyectos/
├── Recursos/
└── README.md
```

> La estructura del repositorio puede actualizarse a medida que se desarrollen nuevas actividades durante el semestre.


## Objetivos del repositorio

- Organizar el desarrollo de los proyectos del curso.
- Documentar el proceso de aprendizaje.
- Mantener un historial de cambios mediante Git.
- Centralizar todos los ejercicios y entregas en un solo lugar.
